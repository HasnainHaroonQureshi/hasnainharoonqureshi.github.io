Deep Learning-Based Structural Crack Detection, Segmentation, Width Estimation & Severity Classification

Using YOLOv11s-seg

Developed a state-of-the-art AI system for automated structural health monitoring. The model detects, segments, and classifies cracks in concrete structures — estimating crack width and severity to assist engineers in maintenance decisions.

  • Instance segmentation using YOLOv11s-seg architecture
  • Real-time crack width estimation via pixel-to-mm calibration
  • Multi-class severity classification (Hairline, Fine/Minor, Moderate, Severe)
  • Web platform for desktop deployment
  • Android application for field use
